# BAMSE Childhood Profiling

Code accompanying the manuscript:
“Longitudinal protein profiling of blood during childhood into early adulthood”

## Overview
This repository contains the code used to analyze longitudinal plasma proteomics data from the BAMSE cohort, with samples collected at ages 4, 8, 16, and 24 years.

## Analyses
The analyses include:
- Overview of sample distribution and PCA across follow-ups
- Paired protein comparisons between consecutive follow-ups (4 vs 8, 8 vs 16, 16 vs 24)
- Sex-stratified and sex-associated protein analyses at each follow-up
- Trajectory clustering of age-associated proteins across childhood and early adulthood
- Gene Ontology enrichment analyses of protein clusters and sex-associated proteins
- Covariate-adjusted models including BMI, body fat, smoking/snuff use, and blood cell counts
- Generation of manuscript figures, tables, supplementary tables, and supplementary data files
